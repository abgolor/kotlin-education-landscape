# Methodology

## Data Collection
Data was collected from:
- University course catalogs
- Computer science syllabi
- Bootcamp curricula
- Online learning platforms
- Official Kotlin education resources

## Data Classification

Programs were categorized into:

- **Java-first**: Java is the primary language
- **Kotlin-introduction**: Kotlin introduced after Java
- **Kotlin-first**: Kotlin is the primary language
- **Multiplatform-ready**: Kotlin Multiplatform included

## Regional Analysis

Data was grouped into:
- Europe
- North America
- Africa
- Online platforms

## Data Validation

All entries were verified using:
- Official course pages
- Institutional websites
- Public curriculum documentation
