# Kotlin Education Landscape

A data-driven research project analyzing how Kotlin is taught across universities, bootcamps, and online learning platforms worldwide.

---

## Overview

Kotlin has become the official language for Android development and is expanding into backend and multiplatform ecosystems. However, its adoption in education remains uneven.

This project investigates:

- Where Kotlin is taught as a primary language  
- Where Java is still dominant  
- How Kotlin is introduced in academic and training environments  
- Regional differences in Kotlin adoption  

---

## Objectives

- Map Kotlin adoption across different education systems  
- Identify gaps between academia and industry practices  
- Analyze regional trends in Kotlin usage  
- Provide insights to improve Kotlin education globally  

---

## Methodology (Summary)

### Data Collection
Data was gathered from:
- Official university course pages  
- Computer science curricula  
- Bootcamp program descriptions  
- Online learning platforms (JetBrains, Coursera, Udacity)  

### Classification Framework

Each program was categorized as:

- Java-first — Java is the primary teaching language  
- Kotlin-introduction — Kotlin introduced after Java  
- Kotlin-first — Kotlin is the primary language  
- Multiplatform-ready — Includes Kotlin Multiplatform  

### Regional Analysis

Programs were grouped into:

- Africa  
- Europe  
- North America  
- Online platforms  

---

## Key Findings

- Online platforms (JetBrains, Udacity, Coursera) are Kotlin-first  
- Universities are largely Java-first or transitional  
- Bootcamps are bridging the gap between academia and industry  
- Kotlin Multiplatform adoption remains limited in education  

---

## Key Insight

Kotlin education is currently being driven by industry-aligned platforms rather than traditional academic institutions.
